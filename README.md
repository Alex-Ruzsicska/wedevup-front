## Getting started

- Run "git clone git@github.com:Alex-Ruzsicska/wedevup-front.git"
- Navigate to the App's folder
- Run "yarn install"
- Create a .env file based on .env.example
- Run "yarn start"
- Access http://localhost:3000/ on the browser
